This repository contains a quantitative research framework designed to identify and analyze "Structural Specials" within the securities lending market.

By shifting the focus from broad market volume to a high-margin "Value Lending" approach, this project demonstrates how public short interest data can be used to isolate securities that persistently command elevated borrow demand.

The core of this project is a data pipeline that synchronizes bi-monthly Cboe BATS short interest reports with market price action and fundamental data. The analysis identifies a small but highly significant cohort of "Composite Specials" that represent the majority of potential lending revenue.
